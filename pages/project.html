The Project

Introduction

The work presented in this website is part of a project called giCASES whose objective is to create a University-Enterprise Alliance for a Spatially Enabled Society.

This Knowledge Alliance is co-funded by the Erasmus+ programme of the European Union that involves various European universities â€“ and Politecnico di Milano among them - with the purpose of reducing the gap between the knowledge students obtain while studying and the knowledge required once they start working. The idea is that this goal can be achieved through a collaboration with relevant companies presenting real life cases for the students to solve.

For this project, the students of the Geographic Information System course of professor Brovelli teamed up with Epsilon Italia to develop the case study CS5 - Harmonizing data flows in Energy Saving EU policies.

INSPIRE

The cornerstone of the project is the INSPIRE Directive with a focus on how states of the European Union are required to transform their data into an INSPIRE compliant format.

This will lead to the creation of a common spatial data infrastructure, enabling the sharing of spatial information between organisations and countries all over Europe and assisting them in policy-making across boundaries.

The project

The main goal of the project was to harmonize data flows related to energy saving policies for buildings located in the centre of Sondrio, one of the provincial capital of Lombardy region in Italy.

In particular, the two objectives of the work were:

  • perform a transformation of the CENED (Certificazione ENergetica degli EDifici) dataset according to the extended INSPIRE Building 2D core data model;
  • show the CNED dataset of Sondrio together with additional ancillary information using a WebGIS.

The transformation

Before starting the actual work, an understanding of the main goal of the INSPIRE Directive and the study of both the source data model and sets given from CENED and the target data model and schema from INSPIRE were required.

Once those knowledge was acquired, the first step in harmonizing the data was to link the simpler source data model to the more complex target data model, matching attributes and values. In order to visualize the attributes needed from the target data model and the corresponding attributes in the source data model, a mapping table was created and, based on that, we were able to perform the transformation using Hale Studio, an open source data transformation program.

The final output was a GML (Geography Markup Language) file, which can be downloaded here.

The Web GIS

The last part of the project was to create this web site and the belonging web GIS with all its current features, using GeoServer and OpenLayers for publishing the spatial data and putting it on the site. The pages themselves are created using .html scripts, while the .js and .css scripts help the functioning of the page and the map features accordingly.

On the Web GIS in this site, there is a visualization of the original CENED dataset of the buildings selected for this project. There is a selection of 103 buildings in the centre of Sondrio. The boundaries of the province and the building footprint can also be found. The last to vector layers found, are dams, hydraulic regulation networks and geothermal plants in the area. These layers are shown, because they either already provide electric or heat energy, or they have the potential toprovide energy for housing, if built and facilitated. There are also 4 different basemaps to be found.

Main Technologies

QGIS

GeoServer

OpenLayers

hale STUDIO

QGIS is an open source desktop GIS used to visualize the data and clean the dataset to only the selected buildings.

GeoServer is an open source software used to publish data from any major data source using open standards.

OpenLayers is an open source software used to create the web GIS. It can put dynamic maps in any web page.

Hale Studio is an open source data transformation software used to transform the data from CENED to the INSPIRE compliant format.
